Selamat Datang di Dokumentasi API Gempa

API ini menyediakan akses data gempa dari BMKG (Badan Meteorologi, Klimatologi, dan Geofisika)

Data Gempa ini diambil pada Tanggal 17 Oktober 2023

Anda dapat mengakses data gempa terkini dan gempa yang dirasakan menggunakan metode HTTP GET:

  • Mendapatkan Semua Data gempa GET /gempa
